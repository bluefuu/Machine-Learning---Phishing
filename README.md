The dataset is taken from Universiti Malaysia Sarawak. It consists of 30,000 websites, 15,000 legitimate websites and 15,000 phishing websites.
## Features
1. Blacklist
2. IP address
3. @ symbol
4. Long URL
5. Multi Sub-Domains
6. Domain Age
7. Https
8. Cookie
9. Redirection
10. DNS record
11. Web Traffic
12. Google Index
